# CodeAlpha_Flashcard_quiz_app

A modern Flashcard Learning App built with Flutter, Provider, and Hive.

## Features

- Add Flashcards
- Edit Flashcards
- Delete Flashcards
- Favorite Flashcards
- Quiz Mode
- Statistics Dashboard
- Category System
- Local Storage with Hive
- State Management using Provider

## Tech Stack

- Flutter
- Dart
- Provider
- Hive Database
- Material Design

## Description

This project is developed as part of CodeAlpha internship. It demonstrates clean architecture, state management, and local database usage in Flutter.
